How to Enable Oz Applets

How It Works

Applet execution is controlled by MIME types. An applet has the MIME type application/x-oz-application and the filename extension oza.

Information Needed

To make your Web Browser Oz Applet enabled, it needs to know the following bits of information:
  1. The mime type application/x-oz-application.
  2. The file extension oza.
  3. Where the program to execute applets resides (For example, /usr/local/oz/bin/ozengine).

Additionally, you have to have Mozart installed, which is not yet documented here.

Enabling Netscape

  1. Open the preferences window.
  2. Choose the Navigator, Applications dialog.
  3. Press the New button.
  4. Enter the information as sketched bove, as you see it done here.
