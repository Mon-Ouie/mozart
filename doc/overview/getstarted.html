Getting Started

Documentation of the Mozart Programming System

The System Installation Manual

guides you through the steps required to install the mozart distribution on your computer. We make available binary, RPM and source distributions. Sources are also available through our anonymous CVS server.

The Demo Applications Document

gives you a guided tour of a collection of interactive Oz applications. Each application can be invoked directly from the browser simply by clicking. This requires that the Mozart system first be installed on your computer and The System Installation Manual explains how to do this.
